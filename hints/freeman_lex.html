For three numbers it's just checking two conditions. 

def max_of_three(a: int, b: int, c: int) -> int:

    if a >= b and a >= c:
        return a
    if b >= a and b >= c:
        return b
    return c

Using reduce function from built-in functools module which makes the same checking and keep the biggest element on every step. 

from functools import reduce


def max_of_three(a: int, b: int, c: int) -> int:
    
    return reduce(lambda x, y: x if x > y else y, (a, b, c))
