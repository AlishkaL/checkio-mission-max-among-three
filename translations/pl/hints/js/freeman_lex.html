Proszę złapać pomysł 💡! To zadanie jest bardzo proste, zwłaszcza z rozwiązaniem, które Państwu proponuję. Proszę użyć wbudowanej funkcji, aby znaleźć maksimum w sekwencji.

Ok, od czego powinienem zacząć?

Proszę zapoznać się z wbudowaną metodą Math.max() i skorzystać z niej. 

Math.max(a, b, c)
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zwrócić wynik wyrażenia podpowiedzi i zawinąć kod w wyrażenie funkcji. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

function maxOfThree(a: number, b: number, c: number): number {

    return Math.max(a, b, c);
}

Jasne, proszę pobrać!

Dla trzech liczb wystarczy sprawdzić dwa warunki. 

function maxOfThree(a: number, b: number, c: number): number {

    if (a>= b && a>= c) return a;
    if (b>= a && b>= c) return b;
    return c;
}

Używając metody reduce, która wykonuje to samo sprawdzanie i zachowuje największy element w każdym kroku. 

function maxOfThree(a: number, b: number, c: number): number {

    return [a, b, c].reduce((max, curr) => (curr> max? curr: max), Number.MIN_SAFE_INTEGER);
}
