Proszę zapoznać się z wbudowaną funkcją max() i korzystać z niej. 

max(a, b, c)
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zwrócić wynik wyrażenia podpowiedzi i zawinąć kod w wyrażenie funkcji. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

def max_of_three(a: int, b: int, c: int) -> int:

    return max(a, b, c)

Dla trzech liczb wystarczy sprawdzić dwa warunki. 

def max_of_three(a: int, b: int, c: int) -> int:

    if a>= b and a>= c:
        return a
    if b>= a and b>= c:
        return b
    return c

Używając funkcji reduce z wbudowanego modułu functools, która wykonuje to samo sprawdzanie i zachowuje największy element w każdym kroku. 

from functools import reduce


def max_of_three(a: int, b: int, c: int) -> int:
    
    return reduce(lambda x, y: x if x> y else y, (a, b, c))
