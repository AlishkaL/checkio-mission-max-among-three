Bei drei Zahlen werden nur zwei Bedingungen überprüft. 

function maxOfThree(a: number, b: number, c: number): number {

    if (a>= b && a>= c) return a;
    if (b>= a && b>= c) return b;
    return c;
}

Verwenden Sie die Methode reduce, die dieselbe Prüfung durchführt und das größte Element in jedem Schritt beibehält. 

function maxOfThree(a: number, b: number, c: number): number {

    return [a, b, c].reduce((max, curr) => (curr> max? curr: max), Number.MIN_SAFE_INTEGER);
}
