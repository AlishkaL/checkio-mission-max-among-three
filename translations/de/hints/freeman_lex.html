Bei drei Zahlen werden nur zwei Bedingungen überprüft. 

def max_of_three(a: int, b: int, c: int) -> int:

    if a>= b and a>= c:
        return a
    if b>= a and b>= c:
        return b
    return c

Verwenden Sie die Funktion reduce aus dem eingebauten Modul functools, die dieselbe Prüfung durchführt und das größte Element bei jedem Schritt beibehält. 

from functools import reduce


def max_of_three(a: int, b: int, c: int) -> int:
    
    return reduce(lambda x, y: x if x> y else y, (a, b, c))
