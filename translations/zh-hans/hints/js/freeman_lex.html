对于三个数字，只需检查两个条件。 

function maxOfThree(a: number, b: number, c: number): number {

    if (a>= b && a>= c) return a;
    if (b>= a && b>= c) return b;
    return c;
}

使用reduce 方法进行同样的检查，并在每一步中保留最大的元素。 

function maxOfThree(a: number, b: number, c: number): number {

    return [a, b, c].reduce((max, curr) => (curr> max? curr: max), Number.MIN_SAFE_INTEGER);
}
